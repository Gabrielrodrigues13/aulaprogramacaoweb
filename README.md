# aulaprogramacaoweb

Objetivo do projeto:
Consetização do publico sobre a importancia de ajudar uma pessoa necessitada, assim como moradores de rua que podem estar nessa situação por diversos motivos.

Tecnologias utilizadas:
A respeito de softwares utilizados foram: Visual Studio Code para programação do site e GitHub para hospedação e versionamento. Programação utilizada para o projeto foram: HTML, CSS e Java Script.

Para Rodar o Projeto localmente:
É necessario entrar no Github e baixar o repositório ou você pode baixar no link a seguir a versão completa do site: https://sprightly-eclair-3ce567.netlify.app

Estrutura do repositório:
Repositório foi segmentado da seguinte maneira: versões inteiras como v01, v02, v03, e etc. São versões com as maiores mudanças. e versões fracionadas como v01.1, v02.1, e etc. São versões de atualização para corrigir erros na programação do site.

Checklist de melhorias feitas na versão final:

Acessibilidade: Com o HTML semantico e ARIA.
Versionamento: A ultima versão apresenta alguns erro de CSS que não foram resolvidos a tempo de finalizar o projeto
Otimização: Esse site funciona em todos os navegadores conhecidos.
